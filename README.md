## 💡 Key Concepts Learned

Throughout this repository, I explored not only Java syntax but also the reasoning behind Java's design.

Some examples include:

- Why composition is often preferred over inheritance.
- Why `Runnable` is preferred over extending `Thread`.
- The difference between `run()` and `start()`.
- Race Conditions and thread safety.
- `synchronized` vs `volatile`.
- Why `ExecutorService` exists.
- The relationship between `Callable` and `Future`.
- Lambda Expressions and Functional Interfaces.
- How Stream API enables functional-style programming.
